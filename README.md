# FitTrack App

## Author
Emmanuel Chukwuemeka

## Module
Mobile Application Development

## Description
FitTrack is an Android fitness tracking application developed using Kotlin and Android Studio.
The app allows users to track daily activity including steps and calories.

## Features
- User authentication
- Activity dashboard
- Step tracking
- Clean and simple UI

## Technologies Used
- Kotlin
- Android Studio
- MVVM Architecture
- Git & GitHub

## Development Process
The project was developed using an agile approach with iterative development and testing.

## How to Run
Open the project in Android Studio, sync Gradle, and run on an emulator or physical device.
